## 기능 요구사항

### 입력

- [x] 구입 금액을 입력받는다.
- [x] [예외] 잘못된 값 입력 시 Error를 발생시키고 해당 지점부터 다시 입력받는다. (종료 X)
  - [x] [예외] 구입 금액이 1,000원 단위로 나누어 떨어지지 않을 때
- [x] 당첨 번호를 입력받는다.
  - [x] [예외] 입력된 숫자가 6개가 아닐 때
  - [x] [예외] 중복된 숫자가 입력되었을 때
  - [x] [예외] 1부터 45 사이의 숫자가 아닐 때
- [x] 보너스 번호를 입력받는다.
  - [x] [예외] 1부터 45 사이의 숫자가 아닐 때
  - [x] [예외] 당첨 번호 중 중복된 숫자가 있을 때

### 로또 발행

- [x] 구입금액만큼의 로또 개수를 발행한다.
- [x] 발행된 6개의 무작위 로또 번호를 Lotto 클래스에 주입한다.
- [x] 로또 번호는 오름차순으로 정렬한다.
- [x] 발행한 로또 수량 및 번호를 출력한다.

**로또**

- [x] 외부에서 주입받은 로또 번호(#numbers)를 상태로 갖는다.
- [x] [예외] 중복된 숫자가 있을 경우

### 당첨 내역 및 수익률 계산

- [x] 입력받은 당첨 번호 및 보너스 번호와 발행된 로또 번호를 비교하여 당첨 내역을 계산한다.
- [x] 당첨 통계를 출력한다.
- [ ] 당첨 내역을 바탕으로 수익률을 계산한다.
  - [ ] 수익률을 소수점 둘째 자리에서 반올림한다.
- [ ] 총 수익률을 출력한다.
- [ ] 애플리케이션을 종료한다.
