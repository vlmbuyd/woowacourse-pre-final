## 기능 요구사항 분석

### 입력/출력

**입력**

- [x] 자동차 이름을 입력받는다.
  - [ ] 쉼표를 기준으로 구분하여 파싱한다.
- [ ] 시도할 횟수를 입력받는다.

**출력**

- [ ] 매 라운드의 실행 결과를 출력한다.
- [ ] 최종 우승자를 출력한다.

<br>

### 기능

- [ ] 라운드별로 자동차를 순회하며 게임을 진행한다.
  - [ ] 자동차의 전진 여부를 구한다.
  - [ ] 전진하는 자동차일 때 이동 거리를 추가한다.

<br>

### 에러

- [x] 입력되지 않았을 때
- [ ] 쉼표 이외의 구분자로 구분되었을 때
- [ ] 쉼표가 2개 이상일 때
- [x] 이름이 6자 이상일 때

<br>

### 엣지 케이스

- [ ] 우승자가 없을 때
